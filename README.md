## 📖 Sobre

O **IA Consciente** é uma aplicação web desenvolvida com **HTML5, CSS3 e JavaScript** que incentiva o uso crítico, ético e responsável da Inteligência Artificial.

A plataforma auxilia estudantes, professores e usuários em geral a identificar possíveis limitações, imprecisões e padrões comuns encontrados em conteúdos gerados por IA, incentivando sempre a verificação das informações antes de utilizá-las.

Desenvolvido como atividade de **Extensão Acadêmica** da disciplina **Desenvolvimento Front-end**, o projeto busca contribuir para a educação digital, o combate à desinformação e o fortalecimento do pensamento crítico diante das tecnologias baseadas em IA.

# ✨ Funcionalidades

## 🔍 Detector de Possíveis Alucinações

Permite ao usuário colar um texto gerado por Inteligência Artificial para análise automática.

O sistema identifica padrões frequentemente associados a conteúdos potencialmente duvidosos, como:

* 📊 Estatísticas sem fonte;
* 📅 Datas específicas sem referência;
* 💬 Citações genéricas;
* ❗ Afirmações absolutas;
* 🗣️ Linguagem excessivamente persuasiva;
* ✨ Excesso de adjetivos.

Ao final da análise, são exibidos alertas e recomendações para incentivar a validação das informações.

---

## 🤖 Humano × Máquina

Inspirado no **Teste de Turing**, este desafio apresenta dois textos:

* 👤 Um produzido por um humano;
* 🤖 Outro produzido por Inteligência Artificial.

Após a escolha do usuário, o sistema revela a resposta correta e destaca características linguísticas presentes em cada texto.

---

## 📈 Índice de Confiabilidade

Após a análise, a aplicação gera uma pontuação baseada nos padrões encontrados no texto.

O objetivo é oferecer um indicador que incentive a verificação das informações antes de sua utilização.

---

## 💡 Dicas Educativas

Ao final de cada análise, a plataforma fornece recomendações sobre:

* 🔍 Como verificar fontes;
* 📚 Como validar conteúdos produzidos por IA;
* ✅ Boas práticas no uso da Inteligência Artificial;
* ⚠️ Riscos da confiança excessiva em conteúdos automatizados.

# 🚀 Como Executar

Clone o repositório:

```bash
git clone https://github.com/davifigrd/IA-consciente.git
```

Entre na pasta do projeto:

```bash
cd IA-consciente
```

Abra o arquivo **`index.html`** em qualquer navegador moderno.

> 💡 Não é necessário instalar dependências ou configurar servidor.
